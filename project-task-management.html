Project Overview

A comprehensive task management application built with Flutter that helps users organize their daily tasks, set priorities, and track progress. The app features a clean, intuitive interface with real-time synchronization across devices.

The application uses Firebase for backend services including authentication, real-time database, and cloud storage. State management is handled using Provider pattern, ensuring smooth performance and efficient data flow throughout the application.

Key Features

Firebase Authentication

Secure user login and registration with Firebase

Task Creation & Management

Create, edit, and delete tasks with ease

Priority Levels

Set priority levels for tasks (High, Medium, Low)

Real-time Sync

Tasks sync in real-time across all devices

Categories & Tags

Organize tasks using categories and tags

Progress Tracking

Track completion status and view statistics
