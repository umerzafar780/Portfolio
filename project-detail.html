Project Overview

A voice translation application built with Flutter that enables users to translate text and speech between multiple languages. The app features a clean, intuitive interface with real-time translation capabilities and seamless language switching.

The application uses Firebase for user authentication and stores user preferences and translation history. It integrates with translation APIs to convert text and voice input between various languages, making communication across language barriers effortless and instant.

Key Features

Firebase Authentication

Secure user login and registration with Firebase

Text Translation

Translate text between multiple languages instantly

Voice Input

Speak and translate your voice to another language

Multi-Language Support

Support for numerous languages worldwide
