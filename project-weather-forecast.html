Project Overview

A modern weather forecast application built with vanilla JavaScript that provides real-time weather information for any location worldwide. The app features a clean, responsive design with dynamic backgrounds that change based on weather conditions.

The application integrates with Weather API to fetch current weather data, 5-day forecasts, and hourly updates. It includes features like location search, geolocation support, and detailed weather metrics including temperature, humidity, wind speed, and more.

Key Features

Real-time Weather Data

Get current weather conditions for any location

5-Day Forecast

View detailed weather forecasts for the next 5 days

Location Search

Search weather by city name or coordinates

Geolocation Support

Automatically detect user's location for weather data

Dynamic Backgrounds

Background changes based on current weather conditions

Responsive Design

Works seamlessly on all devices and screen sizes
